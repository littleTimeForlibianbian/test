2018-08-18网卡

中新网黄石9月11日电 (梁婷 南艳群 丁捷菲)第八届中国创新创业大赛(湖北赛区)决赛11日在湖北黄石举行，经过角逐，排序靠前的22家成长组企业和9家初创组企业将代表湖北进军全国行业总决赛。 中国创新创业大赛是一项以“科技创新、成就大业”为主题的全国性双创赛事。赛事分为地方赛和全国行业总决赛两个赛程。 本次大赛由科技部火炬高技术产业开发中心指导，湖北省科技厅、湖北省发改委、湖北省教育厅、湖北省财政厅、湖北省人社厅、共青团湖北省委共同主办。 自5月份大赛启动以来，湖北赛区共收到778家企业报名参赛，经过两个月的评审角逐，最终有来自电子信息、先进制造、生物医药、新材料、互联网、新能源及节能环保六大行业领域的53个参赛项目进入湖北赛区决赛。
